# Telegram Random Phrase Bot
A simple Telegram bot that selects a random answer from the written words.

## Features
- "Давай поговорим" button
- Random responses from a predefined list of phrases
- Button appears again after each response

> This is a training repository. The code is simple and made for practice.

# Telegram Meow Random Bot
Простой Telegram-бот котрый выбирает рандомный ответ из написанных слов.

## Функционал
- Кнопка "Давай поговорим"
- Случайные ответы из заранее заданного списка слов
- Кнопка появляется снова после каждого ответа

> Репозиторий тренировочный, код простой, для практики.
